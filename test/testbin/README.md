### 测试流程

* 根据实验指导的要求, 使用你实现的语言写出源代码.
* 使用**你的编译器**根据源代码生成目标文件.
* 从目标文件生成可执行文件.
* 将你的可执行文件作为命令行参数提供给测试程序, 检查你的编译器实现是否正确.

```bash
# 例子
./linux/amd64/test_prime ./prime # 如果你的可执行文件名为prime并且在当前目录
```
